Latihan Soal Limit Aljabar

Soal 1: $$\lim_{{x \to 2}} \frac{{3x - 6}}{{x + 2}} = ?$$

Hint: Substitusi nilai \(x\)
Pembahasan:
Masukkan nilai \(x = 2\) ke dalam fungsi maka:
$$\lim_{{x \to 2}} \frac{{3(2) - 6}}{{2 + 2}} = \frac{{0}}{{4}} = 0$$

Soal 2: $$\lim_{{x \to 4}} \frac{{3x^3 - 48x}}{{x^2 - 16}} = ?$$

Hint: Samakan nilai dengan penyebut
Pembahasan:
Keluarkan \(3x\) maka:
$$\lim_{{x \to 4}} \frac{{3x(x^2 - 16)}}{{x^2 - 16}} = \lim_{{x \to 4}} 3x$$
Masukkan nilai \(x = 4\):
$$\lim_{{x \to 4}} 3x = 3(4) = 12$$

Soal 3: $$\lim_{{x \to 2}} \frac{{4 - x^2}}{{3 - \sqrt{{x^2 + 5}}}} = ?$$

Hint: Kali akar sekawan
Pembahasan:
Kali dengan akar sekawan maka:
$$\lim_{{x \to 2}} \frac{{4 - x^2}}{{3 - \sqrt{{x^2 + 5}}}} \cdot \frac{{3 + \sqrt{{x^2 + 5}}}}{{3 + \sqrt{{x^2 + 5}}}}$$
$$= \lim_{{x \to 2}} \frac{{(4 - x^2)(3 + \sqrt{{x^2 + 5}})}}{{9 - (x^2 + 5)}}$$
$$= \lim_{{x \to 2}} (3 + \sqrt{{x^2 + 5}})$$
Masukkan nilai \(x = 2\):
$$3 + \sqrt{{2^2 + 5}} = 3 + \sqrt{9} = 3 + 3 = 6$$
